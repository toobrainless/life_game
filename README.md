# Conway's Game of Life

Реализация игры "Жизнь" в цикличном пространстве на Python

# Запуск

```sh
python main.py
```

## Управление
1. "S" - начать/приостановить процесс жизни
2. "Q" - выйти
3. "C" - очищает полe
4. "R" - генерирует случайную популяцию
5. ЛКМ - меняет состояние клетки на обратное
